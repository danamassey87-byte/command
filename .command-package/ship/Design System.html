Density

Default is cozy — 14px body, 24px card padding. Admin and data-dense pages drop to tight: 13px body, 12px card padding, 6px row spacing. Dana can toggle via Tweaks.

What to avoid

  • Gradient backgrounds, neon accents, shadow blooms.
  • Stock iconography — use placeholders or real product photos.
  • Cold color pairings (blue/gray) — the entire system is warm.
  • Sentence case buttons that try to be conversational ("Let's save it") — our tone is direct.
  • Fake data slop — if a number doesn't teach Dana something, cut it.
